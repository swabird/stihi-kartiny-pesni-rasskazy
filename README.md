# Стихи, картины, песни и рассказы

Личный сайт-архив для стихов, картин, песен, видео, рассказов и небольших
заметок о жизни.

## Структура

- `index.html` — главная страница сайта.
- `styles.css` — оформление.
- `assets/` — визуальные материалы сайта.
- `content/poems/` — полные тексты стихотворений.
- `content/stories/` — рассказы.
- `content/art/` — фотографии картин.
- `content/videos/` — описания песен и видео.

## Как открыть

Открой `index.html` в браузере.

## Что заменить в первую очередь

- Название сайта и подпись в шапке.
- Почту в нижней части страницы.
- Заготовки стихов, рассказов, картин и видео на настоящие материалы.
